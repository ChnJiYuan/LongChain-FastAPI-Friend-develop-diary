# 2025.12.2 Development Log (LangChain + FastAPI Chat Companion)




🚀 第一阶段：LangChain 基础入门（打地基）
首先，您需要掌握 LangChain 的核心概念和模块，因为 LangGraph 是建立在 LangChain 的组件之上的。

1. 理论学习与环境准备
官方文档： LangChain 官方文档是最好的起点，它提供了概念介绍和最新的 API。

资源： LangChain 官方文档（Python 或 JavaScript/TypeScript）。

安装与配置：

安装 Python 环境和 pip install langchain。

获取并设置您的 API Key（如 OpenAI/Gemini/Anthropic 等），并将其配置为环境变量。

核心概念： 重点理解 LangChain 的六大核心模块：

Models: 如何连接和配置不同的 LLM。

Prompts: 提示词模板和输入/输出解析器。

Chains: 如何将多个组件（模型、提示词、解析器）连接成一个序列。

Retrieval: 如何加载、拆分、嵌入数据，并使用向量存储。

Agents: 什么是 ReAct 模式，以及如何赋予 LLM 工具（Tools）。

Memory: 如何在多轮对话中存储和检索历史记录。

2. 实践项目：LangChain Chains & RAG
第一个 Chain： 编写一个简单的 Chain，实现“接收用户输入 -> 格式化提示词 -> 调用 LLM -> 解析输出”的过程。

构建 RAG 应用： 按照官方教程，构建一个简单的检索增强生成 (RAG) 应用。这是 LangChain 最常见的用途。
